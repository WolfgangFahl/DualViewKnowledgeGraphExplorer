# Setup
## Install RISE (Jupyter Slideshow Extension)
```bash
conda activate base
# Install RISE in the conda environment
conda install -c conda-forge rise
```
# Presentations
```bash
conda activate base
```
## Dual View Knowledge Graph Explorer
```bash
jupyter notebook TurnYourStuffIntoResources.ipynb
```
## Query Objects
```bash
jupyter notebook QueryObjectsPresentation.ipynb
```
